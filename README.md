# Browser Pilot — Releases

Distribution shelf for the Browser Pilot Cockpit (source is private).

Download via https://haffner.tech/browser-pilot — assets use stable, version-free names
(`BrowserPilot-Setup[_{agent}].exe`) so `releases/latest/download/…` always serves the
newest build. All assets in a release are identical bytes; the filename carries the
agent seed for the setup wizard.
